Thank you for downloading Chrono Marker, the intuitive stopwatch and time logger gadget!

Chrono Marker requires the following libraries in order to function: .Net 4.0 or Mono equivalent, Gtk# libraries, Gtk+ libraries
Please install them in order ro run Chrono Marker.

Chrono Marker license can be read in the program's About dialog and the LICENSE.TXT file, which should have been distributed with it.

For more information about Chrono Marker and future updates please visit it's Google Code page: http://code.google.com/p/chrono-marker/
